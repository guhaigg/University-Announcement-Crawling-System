# 大学官网公告爬取网页程序

一个本地网页程序，支持：

- 每日定时爬取院校官网公告
- 自助扫描候选链接后，手动勾选并确认爬取正文
- 支持“研究生调剂公告”模式（可匹配研究生招生入口，并可扩展到学院层级）
- 自定义院校官网链接、公告页链接、关键词
- 自助区自动保存“最近十所院校”配置，可一键套用
- 无法查询时先“核对链接”，从候选链接中点选
- 输出 Markdown 报告文件（保存在 `outputs/`）
- 每个定时任务可随时“暂停/开启每日爬取”
- 已内置登录鉴权（未登录无法调用 API）

## 目录结构

- `server.js`: 后端服务、定时任务、抓取和导出逻辑
- `public/`: 前端页面
- `data/config.json`: 任务配置
- `outputs/`: 生成的 Markdown 文件

## 运行方式

1. 安装 Node.js 18+（建议 20+）
2. 在项目根目录执行：

```bash
npm install
npm start
```

3. 浏览器打开：

```text
http://localhost:3000
```

## 登录说明

- 首次启动会自动创建默认管理员账号：
  - 用户名：`admin`
  - 密码：`admin123456`
- 建议在生产环境通过环境变量覆盖默认账号：

```bash
DEFAULT_ADMIN_USER=your_admin
DEFAULT_ADMIN_PASSWORD=your_strong_password
COOKIE_SECURE=true
```

## 使用说明

1. 在“每日爬取任务”中填写：院校名称、官网链接、抓取模式（普通/研究生调剂）、关键词和每日抓取时间。
2. 若不确定公告链接，点“核对公告链接”，从候选列表点击填充。
3. 保存后会进入任务列表，点击“扫描候选公告”。
4. 在“候选公告确认区”勾选要抓取正文的链接，点击“确认爬取并生成文件”。
5. 调剂模式下可开启“学院层级页面”开关，抓取校级+学院调剂相关内容。
6. “自助爬取（一次性）”同样采用“先扫描候选、再确认抓取正文”的流程，不会创建定时任务；系统会保存最近十所院校配置。
7. 输出文件可在页面底部下载或在线预览。

## 说明

- 不同高校网站结构差异较大，若自动识别公告链接不准确，建议手动填入公告页链接。
- 部分网站有反爬策略，可能出现访问失败，可稍后重试或更换可访问链接。
